High-signal, developer-focused diagnostic checks for WordPress Site Health.

== Description ==

PulseCheck adds targeted, developer-focused diagnostics to WordPress Site Health.

Rather than duplicating existing checks, PulseCheck focuses on surfacing
high-signal, actionable information that helps teams quickly assess
platform and environment health.

All checks are read-only and safe to run in production environments.

Designed to be safe for production and easy to extend.

= Current Checks =

* PHP version compatibility and risk assessment
* WordPress core version currency and update recommendation
